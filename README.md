# docker-redis

Redis 6 + Redis Commander

## Images

- [Redis 6 (Alpine Linux)](https://hub.docker.com/_/redis)
- [Redis Commander](https://hub.docker.com/r/rediscommander/redis-commander/)

## Quickstart

```sh
~$ docker-compose up -d
```

| Service         | Path            |
| --------------- | --------------- |
| Redis           | localhost:6379  |
| Redis Commander | localhost:49001 |
